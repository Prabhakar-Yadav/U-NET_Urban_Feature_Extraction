# Final Results

This folder stores presentation-ready outputs from the final experiments.

## Contents

- `jnpa/` - JNPA result maps, dense-label preview, and summary boards
- `cartosat/` - CARTOSAT result maps, dense-label preview, and summary boards
- `test_data_evaluation/` - summary of predictions on the held-out test-data folder

These files are included to document the final model behavior without committing the full multi-gigabyte runtime results directory.
