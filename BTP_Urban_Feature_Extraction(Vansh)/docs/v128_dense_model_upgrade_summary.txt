V128 DENSE MODEL UPGRADE SUMMARY

Date:
- 2026-03-24

What changed:
- New 128x128 segmentation pipeline created
- Two variants tested for each dataset:
  1. Hard per-pixel pseudo masks
  2. Soft per-pixel probability masks
- Compared against the current platform baseline for the same dataset

--------------------------------------------------
1. JNPA RESULT
--------------------------------------------------

- Previous production JNPA patch-classifier baseline
- Evaluated against the new dense pseudo-mask validation setup

JNPA baseline:

- Mean IoU: 35.89%

JNPA hard U-Net V128:
- Pixel Accuracy: 89.28%
- Mean IoU: 76.05%

JNPA soft U-Net V128:

- Mean IoU: 78.73%

Best JNPA model:
