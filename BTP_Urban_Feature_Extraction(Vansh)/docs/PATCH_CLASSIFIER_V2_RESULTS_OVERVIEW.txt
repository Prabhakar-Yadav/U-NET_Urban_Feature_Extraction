Improved Patch Classifier V2 - Results Overview

Why accuracy improved:
The earlier JNPA model used a U-Net segmentation network, but the pseudo-labels were only patch-level labels.
That mismatch limited the score.

The new V2 pipeline uses:
1. Unsupervised clustering to create pseudo-labels
2. PCA-reduced patch features
3. Random-forest patch classification
4. Full-scene patch-wise reconstruction

This matches the label design much better and gives much higher validation accuracy.

3. Improved CARTOSAT V2 Run
--------------------------------------------------
Folder:
outputs/cartosat_patch_classifier_v2

Main metrics:
- Validation Accuracy: 0.9416
- Validation Macro F1: 0.6358

Note:
The CARTOSAT scene is dominated by water and urban patches.
The terrain class is much smaller and harder to separate cleanly in this monochromatic scene.
So overall accuracy is very high, but macro F1 is lower because terrain is underrepresented.
