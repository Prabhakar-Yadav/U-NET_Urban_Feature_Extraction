Remote Sensing Inference Platform

Upload an image, match it to the right trained model, and get a feature map in minutes.

This platform runs only on trained project models, including the new 128x128 dense U-Net production models, the earlier patch-classifier baselines, and older U-Net comparison checkpoints. It returns a marked image, a segmentation map, class percentage coverage, and the model-fit score used for selection.

What You Get

  • Automatic model selection between the promoted JNPA and CARTOSAT dense models
  • Manual comparison against older patch-classifier and U-Net project checkpoints
  • Color-coded feature detection map and marked preview overlay
  • Percentage area by class for the uploaded scene
  • Downloadable result files for reporting and backend reuse
